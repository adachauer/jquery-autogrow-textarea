jQuery-autogrow-textarea

A jQuery plugin to make textarea be resized automatically to fit its contents.

Download .zip Download .tar.gz View on GitHub

Pros

  • Capable with multibyte characters.
  • Simple to use.

A bit more precise description

Typical plugins that provides similar features with this plugin usually hooks keypress event to observe textarea input. However, under multibyte IME environment, keypress, keydown, keyup events are never invoked until trans literation choices to be fixed. So that I, as a Japanese, decided to write another implementation to do the job.
